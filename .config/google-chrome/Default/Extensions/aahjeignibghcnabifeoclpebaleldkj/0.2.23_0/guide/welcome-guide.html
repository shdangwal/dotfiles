Thanks for installing tabVertikal

This guide help you to understand how tabVertikal work and How to use tabVertikal?

Switch panel position

Paste this url in chrome://settings/appearance#:~:text=side%20panel in chrome's address bar then Select select Show on left

Keyboard shortuts

Open extension's options page to view all Keyboard shortuts Right click on extension's icon and click on Options from popup menu.

Hide url

Open tabVertikal side panel and click on menu icon on top-right corner. Click on Compact mode option from popup menu. keyboard shortcuts: Ctrl+Alt+C

Switch workspace

Open tabVertikal side panel and click on arrow-down icon on middle of top bar. Click on Create workspace option to create new workspace or select existing workspace. All opened tabs are saved in current workspace. Current workspace's tabs are replace with Selected workspace's tabs.

Create grouping rule for auto tab grouping

Open tabVertikal side panel and click on menu icon on top-right corner. Click on tabGroup rules option from popup menu.
tabGroup rules dialog open.
Click on Add rule button to create new rule. For disable auto tab grouping, go to extension's options page. keyboard shortcuts: Ctrl+Alt+R

Enable auto tab suspension

Click on tabVertikal's icon and select Options from popup menu. Extension's options page will open. Switch on Enable Auto tab suspension then select time. tabVertikal will auto suspend tabs which are not using more than selected time. default time is 30 min.

Import/export tab

Open tabVertikal side panel and click on menu icon on top-right corner. Click on Import/export tab option from popup menu.
Import/export dialog open.
EXPORT: select file format and click on export button.
IMPORT: Upload html,json,csv,markdown file. All urls inside file will open keyboard shortcuts: Ctrl+Shift+E
