#Dotfiles

## Pre-requisites

Install following packages before using stow

`brew install git`
`brew install stow`

Create `dotfiles` folder in home directory then clone this repo and use the following command to replicate the changes.

`stow adopt .`

## Dotfiles for following programs

1. zsh
2. neovim
3. oh-my-posh
4. wezterm
